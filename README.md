# ampblocker
 Blocks Finder.app from automatically launching when an Apple device is connected (macOS 10.15+)
 
 ### Building:
 ```bash
 git clone https://github.com/staturnzz/ampblocker.git
 cd ampblocker
 chmod +x build.sh
 ./build.sh
 ```
 
 
 ### Usage:
 ```bash
 ./ampblocker <start/stop> (optional force start/stop) -f
 ```
